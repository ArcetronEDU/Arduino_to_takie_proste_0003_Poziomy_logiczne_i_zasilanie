"0003 Poziomy logiczne i zasialnie"
W filmie przedstawiłem najważniejsze szczegóły dotyczące użycia Arduino Uno i ESP32 aby nie uszkodzić wejść i wyjść pinów cyfrowych, 
nie oglądając tego filmu i pomijając go narażasz na uszkodzenie twojej płytki programowania.
To jeden z trudniejszych filmów na start ale bardzo istotny dla każdego nowego programisty, dużo danych na raz, ale jest i można do niego wracać w razie potrzeb.

Materiały dodatkowe do filmów "Arduino to takie proste" (https://www.youtube.com/watch?v=Z5_0qcEuFcI)
